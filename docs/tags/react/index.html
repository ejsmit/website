create-react-app & Docker

Tue, Jul 7, 2020 - 300 Words
These instructions are not perfect, but they work for the small webapps I create. Installation npx create-react-app my-app cd my-app rm -r node_modules dockerfile.dev used to host a dev server, and run tests. Just use docker-compose up to start. No need to manually build container. FROMnode:lts-alpineRUN mkdir -p /home/node/app/node_modules \ && chown -R node:node /home/node/appWORKDIR/home/node/appUSERnodeCOPY package.json .COPY yarn.lock .RUN yarn installCOPY --chown=node:node . .EXPOSE3000CMD ["yarn", "start"]dockerfile Build the webapp in a temporary container, and then create a production nginx server to host the app.
