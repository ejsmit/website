Mermaid charts

Using mermaid to draw html charts in a Hugo website

7 Apr, 2020
  1. My Notebook
  2. Website
  3. Mermaid charts

Shortcodes

Mermaid charts are created using Hugo shortcodes. Here is a simple example.

{{< mermaid align="left" >}}
graph LR
    Start --> Stop
{{< /mermaid >}}
graph LR Start --> Stop

See more at https://mermaid-js.github.io/mermaid/#/
